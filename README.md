# ЯПрофи

## Для проверки проекта:

### Запустить самостоятельно

1. Необходима .Net Core 5.0>= (SDK)

2. Для билда серверной части выполните команды в папке приложения (YaProfiUrfu)

   ```
   dotnet restore
   dotnet build
   ```

3. Для запуска скомпилированного приложения в папке выполните команду:
   ```
   dotnet run
   ```
   Будет запущено на локальной бд (inMemory), на локальном адресе с портом 5001 (порт можно поменять в
   launchSettings.json) `https://localhost:5001/`

## Для проверки функционала:

1. Воспользуйтесь swagger-ом (/swagger)
2. Функционал N "Если заголовок не указан - то вместо него возвращать первые N символов текста заметки, где N - число,
   задаваемое в конфигурационном файле" можно заменить в appsetings.json файле. Переменная "TakeDefaultN": 10
